## Prerequisites

- Minecraft forge 1.8.9
- Java version 8

## Install

```java
Move the Mod to your mods folder
```

## Usage

```sh
/melodyfinder toggle
/melodyfinder debug
/melodyfinder reload
/melodyfinder recheck
```
